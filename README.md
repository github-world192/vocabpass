超級單字app

一個準備在安卓商店上架的單字學習應用程式，支援跨平台開發，並結合 Flutter (Dart)、C++ 等多種技術。
本專案致力於幫助使用者輕鬆背單字、測驗記憶力，提升英文能力。
特色功能

    單字記憶練習
    單字測驗/小遊戲
    支援離線使用
    優雅簡潔的 UI
    多語言技術架構（Flutter/Dart、C++、Swift）
    準備上架 Google Play

技術棧

    Flutter/Dart：跨平台 app 開發
    C++/CMake：部分底層運算/效能優化
    Swift：iOS 相關支援
    HTML：部分靜態內容或嵌入式頁面

安裝與執行
Flutter 開發環境

    安裝 Flutter SDK
    下載本專案
    bash

git clone https://github.com/github-world192/efgefg.git
cd efgefg

安裝依賴
bash

flutter pub get

執行 App
bash

    flutter run

C++/CMake 組件

如需編譯底層 C++ 模組，請先安裝 CMake 及相關 C++ 編譯器。具體請參見 cpp/ 目錄內說明或 CMakeLists.txt 設定。
